Wireframe Renderer

Takes an input file containing a 2D map of heights, and renders each elements connections to the left, right, forward and back.

Optionally takes colour values with the heights.

The scene is rotatable with the W, A, S, D keys, movable with the arrow keys, and zoomable with the + and - keys.
